# Feature Review

Use this skill to review a canonical `feature.md` before planning or execution depends on it.

## Execution

1. Use `../memory-bank-review-feature.md`.
2. If any criterion fails, fix the feature and review again through `review-loop.md`.
3. If independent confirmation is needed, trigger Claude Code MCP through `review-loop.md`.

## Preserve

- template fit;
- traceability;
- executable verification;
- lifecycle-gate readiness;
- explicit states, constraints, and failure modes when material.
