# Use-Case Review

Use this skill to review a governed use case before multiple features rely on it.

## Execution

1. Use `../memory-bank-review-use-case.md`.
2. If any criterion fails, fix the use case and review again through `review-loop.md`.
3. If independent confirmation is needed, trigger Claude Code MCP through `review-loop.md`.

## Preserve

- stable scenario boundary;
- explicit trigger and actor;
- complete main and alternate flows;
- honest traceability.
