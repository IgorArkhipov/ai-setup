# Evaluation checklist

Evaluate the proposed solution against the following:

## 1. Goal Fit
- Does the solution actually solve the requested task?
- Has the task been replaced with a similar but different one?

## 2. Acceptance Criteria
- Which acceptance criteria are clearly satisfied?
- Which are missing, partially satisfied, or unverified?

## 3. Evidence
- What artifacts support the conclusion?
- What is missing to be confident?

## 4. Risks
- What could be broken?
- Where is human review still needed?

## 5. Decision
- Accept / Revise / Escalate
- What exactly should be fixed next?
