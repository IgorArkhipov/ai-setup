AgentScope is a standalone TypeScript tool in `tools/agentscope/` for discovering and managing local AI-agent configuration across Claude Code, Codex, and Cursor. It inventories skills, configured MCP servers, and other provider-managed items across global and project scopes, normalizes them into a shared model, and exposes that state through a CLI.

The project is built around safe, reversible change management. AgentScope plans mutations before applying them, uses advisory locking and state checks to avoid unsafe writes, stores backups, audit history, vault metadata, and project-scoped discovery snapshots under `~/.config/agentscope/`, and supports restore flows so users can enable or disable supported items without manually editing provider files.

The current scope stays intentionally narrow: it focuses on honest discovery, persisted discovery snapshots, and reversible management of skills and configured MCPs. Claude, Codex, and Cursor have verified writable coverage for their supported slices. A dashboard or local stdio MCP surface may return later, but neither is implemented in `tools/agentscope` today.
