AgentScope is a standalone TypeScript tool in `tools/agentscope/` for discovering and managing local AI-agent configuration across Claude Code, Codex, and Cursor. It inventories skills, configured MCP servers, and other provider-managed items across global and project scopes, normalizes them into a shared model, and exposes that state through both a CLI and a terminal dashboard.

The project is built around safe, reversible change management. AgentScope plans mutations before applying them, uses advisory locking and state checks to avoid unsafe writes, stores backups and snapshot history under `~/.config/agentscope/`, and supports restore flows so users can enable or disable supported items without manually editing provider files. The dashboard is not a separate control path: it stages changes visually, then delegates to the same headless discovery and mutation engine used by the CLI.

AgentScope also provides a local stdio MCP server so Claude, Codex, and Cursor can use structured inventory, planning, apply, restore, and doctor tools through natural-language workflows after the user registers it as an MCP endpoint. The current scope stays intentionally narrow: it focuses on honest discovery plus reversible management of skills and configured MCPs, while broader plugin and tool installation lifecycle work remains explicitly out of scope.
