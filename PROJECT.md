AgentScope is a standalone TypeScript tool in `tools/agentscope/` for discovering and managing local AI-agent configuration across Claude Code, Codex, and Cursor. It inventories skills, configured MCP servers, modern configuration surfaces such as agent files/hooks/settings/plugin manifests or config declarations, and other provider-managed items across global and project scopes, normalizes them into a shared model, and exposes that state through a CLI.

The project is built around safe, reversible change management. AgentScope plans mutations before applying them, uses advisory locking and state checks to avoid unsafe writes, stores backups, audit history, vault metadata, and project-scoped discovery snapshots under `~/.config/agentscope/`, and supports restore flows so users can enable or disable supported items without manually editing provider files.

The current scope stays intentionally narrow: it focuses on honest discovery, persisted discovery snapshots, reversible management of verified writable skills, configured MCPs, and agent files, read-only visibility for other modern provider surfaces, and a local stdio MCP surface over the same headless core. Claude, Codex, and Cursor have verified writable coverage for their supported slices. A dashboard remains deferred and is not implemented in `tools/agentscope` unless future scope revives it.
