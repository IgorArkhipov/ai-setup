# Базовый сетап для курса AI driven development

Подходит для базовых Linux-дистрибутивов (Ubuntu-tested) и MacOS

## Автоматическая установка

```
make
```

## Что произойдет при установке

1. В систему установится mise
2. mise установит последнюю node
3. npm установит необходимые cli и skills для claude/codex
